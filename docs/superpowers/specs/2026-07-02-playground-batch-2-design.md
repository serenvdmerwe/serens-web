# Playground batch 2: five new experiments

Date: 2026-07-02. Issue: #28. Branch: `feat/playground-batch-2`.

## Goal

Grow the playground from one demo (Florida Risk Explorer) to six, chosen to show range:
accessibility craft, interaction polish, AI literacy, WordPress depth plus teaching, and
data wrangling plus animation. The playground is the proof shelf for the four service
areas, aimed at founders, marketers, and product teams.

## Delivery model (all five)

Identical to the Florida Risk Explorer:

- One self-contained HTML file per demo in
  `wordpress/wp-content/themes/serensweb-child/assets/playground/`.
  Inline CSS and JS, zero frameworks, zero plugins, zero build step.
- Linked from a `play-card` on the playground page (`patterns/playground.php`),
  opening in a new tab (`target="_blank" rel="noopener"`). The coming-soon
  placeholder card is removed once all five ship.
- Mobile friendly from the start (the Florida demo needed a retrofit; these do not).
- Writing rules apply: no em or en dashes, no double hyphens in prose, no emoji,
  first person voice where copy speaks as Seren, only provable claims.
- Fonts: system stacks, or the theme's self-hosted Geist and Geist Mono referenced
  by absolute path (`/wp-content/themes/serensweb-child/assets/fonts/`). No CDNs.
- Each demo commits to its own distinct aesthetic so the shelf shows visual range too.

## The five demos

### 1. Contrast, the game (`contrast-game.html`)

Guess whether a text and background color pair passes WCAG AA (4.5:1 for normal text)
before the timer runs out. Ten rounds per run. After each guess the real contrast ratio
and the pass or fail verdict are revealed, with the relative luminance math available.
Difficulty ramps: early pairs sit far from 4.5:1, later pairs hug the boundary.
Pairs are generated by rejection sampling toward a target ratio band so hues vary.
Score, best streak, and accuracy persist in localStorage. Dark, game-show styling;
the UI itself must be AA compliant, which is the joke and the point.

### 2. Dev typing test (`dev-typing-test.html`)

Type real code snippets (PHP, JS, CSS, drawn from the kind of code that runs this site)
in Geist Mono. Per-character diffing with correct, incorrect, and pending states,
live WPM ((correct chars / 5) / minutes) and accuracy, Enter for newlines with leading
indentation auto-skipped. Results card with WPM, accuracy, language; best score in
localStorage. Terminal aesthetic: near-black, phosphor green accents, blinking caret.

### 3. AI feature cost estimator (`ai-cost-estimator.html`)

Pick a use case preset (support chatbot, document summarizer, semantic search assist),
then adjust monthly interactions, average input tokens, and average output tokens with
sliders. Live monthly cost cards for Claude Haiku 4.5 ($1 in, $5 out per MTok),
Claude Sonnet 5 ($3 in, $15 out), Claude Opus 4.8 ($5 in, $25 out), and Claude Fable 5
($10 in, $50 out), plus a batch processing toggle (half price) and a prompt caching
slider (cached share of input billed at roughly a tenth of the input rate).
Cost = volume x (in tokens x in rate + out tokens x out rate), displayed per month with
a per-interaction figure. Fine print: prices as of July 2026, estimates only, link to
official pricing. Light, calculator-like styling with tabular numerals.

### 4. How a WordPress page loads (`wp-page-load.html`)

A scrollytelling explainer. A sticky SVG pipeline diagram (browser, DNS and TLS, web
server, page cache, PHP and WordPress boot, database and object cache, template render,
response, browser parse and paint) stays pinned while step cards scroll past; each step
highlights its stage on the diagram via IntersectionObserver and CSS transitions.
Includes the cache-hit shortcut path and closes with what this means for a client site
(caching layers, lean themes). Respects prefers-reduced-motion. Editorial dark styling,
serif display headings. Mobile: diagram pins to the top, steps scroll below.

### 5. Hurricane tracks time machine (`hurricane-tracks.html`)

Real NOAA HURDAT2 Atlantic best-track data, processed offline into a compact embedded
JSON: hurricanes from 1975 through the latest full season, 6-hourly points thinned and
rounded, storm name, year, and peak Saffir-Simpson category retained. An equirectangular
SVG basin map (coastline generated from Natural Earth public domain data) with a year
scrubber and play button; tracks draw on with stroke-dashoffset animation, colored by
category. Tapping or hovering a track shows name, year, and peak intensity. Deep-ocean
dark styling. Data source credited on the page. If the raw data cannot be fetched and
verified, this demo does not ship with fabricated tracks; it waits.

## Playground page

`patterns/playground.php` gains one card per demo (title, two-sentence description,
tech chips, link-out button) and drops the coming-soon card. `theme.css` gains small
`play-card__viz` color variants so the six cards are visually distinct on the shelf.

## Testing

Per demo: banned-character lint, page serves 200 via curl on serensweb.test, markup
sanity checks (key hooks present), manual check in the browser by Seren. Deploy to
production by scp of changed files after merge, then curl verification on serensweb.com.
