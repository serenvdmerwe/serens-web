Design Token Re-themer

Eight tokens in, a whole page out. Nothing on the artboard has a hardcoded style.

Fernway, the page on the artboard, is a fictional walking app I made up for this demo. Every color, font, size, corner, and gap it uses is derived from the eight tokens below, the way I build real sites: decisions live in tokens, pages just read them. My own site re-themes from a single accent token; this is the same idea with all the dials exposed.
