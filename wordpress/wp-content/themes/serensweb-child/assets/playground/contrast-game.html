How the ratio is computed

Each color is converted to relative luminance: sRGB channels are linearized, then weighted as L = 0.2126 R + 0.7152 G + 0.0722 B. The contrast ratio is (L1 + 0.05) / (L2 + 0.05) with the lighter color on top. WCAG AA requires at least 4.5:1 for normal text and 3:1 for large text; this game tests the normal-text bar.
